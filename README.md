# Kala Digree College - Login Page 🎓

Yeh ek simple aur clean **Login Page** hai jo maine HTML aur CSS ka use karke banaya hai. Isme maine modern design aur interactive hover effects ka use kiya hai.

## ✨ Features
- **Responsive Design:** Yeh desktop par badhiya dikhta hai.
- **Hover Effects:** Inputs aur Login button par smooth animations hain.
- **Clean UI:** Eye-catching colors (#e0ffff) aur rounded corners ka use kiya gaya hai.

## 🛠️ Tech Stack Used
- **HTML5:** Structure ke liye.
- **CSS3:** Styling, Flexbox, aur Transitions ke liye.

## 🚀 How to see this project?
1. Is repository ko clone karein ya zip file download karein.
2. `index.html` file ko apne browser mein open karein.
